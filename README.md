FAPI – Objednávkový systém (testovací úkol)

Tento projekt je jednoduchý objednávkový systém vytvořený jako součást přijímacího řízení do FAPI.
Aplikace je postavena na programovacím jazyce Python, HTML a CSS. Je zde použit framework Flask.

Funkce aplikace

Objednávkový formulář
Uživatel může vyplnit:

-Jméno a příjmení / Společnost

-IČO

-Adresu

-Telefon

-Email

-Datum objednávky

-Výběr produktu

-Počet kusů

-Formulář obsahuje HTML validace.

Součástí je také automatický výpočet celkové ceny podle produktu a počtu kusů.
